# 가치봄 제주 RAG 신뢰성 개선 계획

기준일: 2026-07-15

## 목표

현재의 `59/59` 정책·랭킹 회귀검증을 RAG 또는 GPT 성능으로 오해하지 않도록 분리하고, 사람 검수 골드셋과 실제 실행 로그를 기반으로 검색 품질과 설명 근거성을 평가한다.

## 현재 수치의 정확한 의미

- `59/59`: `use_ai=false`로 실행한 5개 상황의 정책·랭킹 회귀검증이다.
- `5/59`: 실제 외부 모델 호출 결과가 아니라 사람이 작성한 무RAG 통제 fixture다.
- GPT-5 mini 저장 응답은 30개 질문의 Before/After 60건이다.
- 저장 응답 자동평가에서 기대 근거 커버리지는 `48.81% → 91.37%`, 사용자 조건 커버리지는 `63.89% → 86.11%`였다.
- 사람 블라인드 검수 90행은 아직 완료되지 않아 GPT 설명 품질은 잠정 결과로만 사용한다.

## 평가 단위 분리

### 검색·추천 평가

동일한 골드셋 질문에 대해 GPT 단독, 룰 기반, 하이브리드 방식의 상위 4개 장소를 비교한다.

- 대표 지표: `Recall@4`
- 배포 차단 조건: 필수 접근성 조건 위반, 제외 장소 추천, 검증되지 않은 사실의 확정적 표현

### 설명 평가

추천 장소를 동일하게 고정하고 룰 템플릿, 근거 없는 GPT-5 mini, 근거 컨텍스트를 받은 GPT-5 mini의 설명만 비교한다.

- 대표 지표: `Grounded Claim Rate`
- 보조 평가: 사용자 조건 커버리지, 사람 선호도, 정확성, 이해도, 안전 이슈율

## Gold Set v1

초기 골드셋은 60개 질문으로 구성한다.

| 영역 | 케이스 수 |
| --- | ---: |
| 휠체어 접근성 | 10 |
| 아이·유모차 | 10 |
| 체력 저하·회복 여행 | 10 |
| 음식·시설 제한 | 8 |
| 날씨·야외 활동 | 8 |
| 복수·충돌 조건 | 6 |
| 오타·모호한 질문 | 4 |
| 적합 장소 없음 | 4 |
| 합계 | 60 |

각 질문은 관련 장소, 허용 장소, 반드시 제외할 장소와 카테고리, 필수 접근성 필드, 필요한 출처 근거, 방문 전 확인사항, 결과 없음 허용 여부를 포함한다. 두 명이 독립 검수하고 불일치는 제3자가 확정한다. 자동 생성 초안은 사람 검수가 끝나기 전까지 최종 골드셋으로 사용하지 않는다.

## 권장 초기 게이트

- `Recall@4 ≥ 0.90`
- `Grounded Claim Rate ≥ 0.95`
- 필수 조건 위반률 `0%`
- 평가 실행 성공률 `100%`
- 사람 검수 커버리지 `100%`

수치 기준은 Gold Set v1 파일럿 결과를 확인한 뒤 확정한다.

## 재현성 요구사항

모든 실험은 `case_id`, 방식, 반복 번호, 모델 ID, 프롬프트 버전, 데이터 기준일, 추천 장소 ID, 사용 근거, 원문 응답, 지연시간, 입력·출력 토큰, 비용을 저장한다. GPT 방식은 같은 질문을 최소 3회 실행한다.

## 발표 원칙

- `59/59`는 정책·랭킹 회귀검증으로만 표시한다.
- `5/59`는 실제 GPT 비교 결과가 아닌 fixture임을 본문에 표시한다.
- 메인 성과 장표에는 `Recall@4`와 `Grounded Claim Rate`만 사용한다.
- 사람 검수가 끝나지 않은 수치는 `자동평가·잠정`으로 표시한다.
- 집계 수치와 함께 케이스별 질문, 통과 항목, 실패 원인을 부록에서 공개한다.
