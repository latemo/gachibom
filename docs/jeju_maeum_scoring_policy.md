# 제주의마음 여행 AI 점수화 정책

## 1. 목적

점수는 관광지를 홍보하기 위한 지표가 아니다.

점수는 사용자의 이동 조건과 장소 접근성 정보가 얼마나 잘 맞는지 설명하기 위한 판단 보조 지표다.

## 2. 총점 구조

총점은 100점 기준이다.

| 평가 항목 | 배점 | 설명 |
| --- | ---: | --- |
| 정보 신뢰도 | 25 | 공식 출처, 운영자 검수, 정보 확인일, 핵심 항목 충족 여부 |
| 이동 부담 적합성 | 25 | 도보 부담, 계단, 경사, 주차장 거리, 체류 시간 부담 |
| 편의시설 충족도 | 20 | 장애인 화장실, 주차장, 휴식 공간, 휠체어 대여 또는 보조기기 정보 |
| 선호 테마 적합성 | 15 | 바다, 숲, 실내, 문화, 조용한 장소 등 사용자가 원하는 경험과 일치 |
| 안전 안내 명확성 | 15 | 주의사항, 확인 필요 항목, 대체안, 보호자 메시지 제공 여부 |

## 3. 등급 기준

| 점수 | 등급 | 의미 |
| ---: | --- | --- |
| 85~100 | A | 매우 적합. 그래도 방문 전 확인 필요 |
| 70~84 | B | 대체로 적합. 일부 항목 확인 필요 |
| 50~69 | C | 조건부 가능. 보호자 확인과 대체안 필요 |
| 30~49 | D | 이동 부담 또는 정보 부족이 큼 |
| 0~29 | F | 현재 조건에서는 추천 부적합 |

## 4. 강제 감점 규칙

아래 조건은 총점에서 강제 감점한다.

| 조건 | 감점 |
| --- | ---: |
| 출처 URL 없음 | -15 |
| 정보 확인일 없음 | -10 |
| 장애인 화장실 정보 없음 + 사용자가 필수로 요청 | -15 |
| 주차 정보 없음 + 사용자가 가까운 주차를 요청 | -10 |
| 계단/경사 정보 없음 + 휠체어 또는 유모차 사용자 | -20 |
| 도보 부담 높음 + 긴 걷기 어려움 | -25 |
| 장시간 야외 체류 가능성 + 체력 저하 또는 회복 중 사용자 | -15 |
| 접근성 상태가 needs_check | 최대 등급 B로 제한 |
| 접근성 상태가 unavailable | 최대 등급 C로 제한 |

## 5. 강제 차단 규칙

아래 조건이면 추천 코스의 핵심 장소로 사용하지 않는다.

- 접근성 판단에 필요한 정보가 전혀 없음
- 사용자가 피해야 한다고 입력한 요소가 명확히 존재함
- 위험 문구 또는 과장 문구가 출처 없이 포함됨
- 의료적 효과를 암시하는 이유로만 추천됨
- 운영자가 부적합으로 표시한 장소

## 6. 정보 최신성 규칙

정보 확인일을 기준으로 신뢰도를 조정한다.

| 확인일 상태 | 처리 |
| --- | --- |
| 6개월 이내 | 감점 없음 |
| 6~12개월 | 정보 신뢰도 -5 |
| 12개월 초과 | 정보 신뢰도 -10, 확인 필요 표시 |
| 확인일 없음 | 정보 신뢰도 -10 |

단, 이 기준은 운영자가 계절성, 공사, 휴무 이슈를 반영해 조정할 수 있다.

## 7. 사용자 조건별 우선순위

휠체어 이용자:

- 계단/경사 정보
- 휠체어 접근 가능 여부
- 장애인 화장실
- 주차장과 입구 거리
- 바닥 상태

회복 중인 여행자 또는 체력 저하 사용자:

- 짧은 도보
- 휴식 가능성
- 혼잡 회피
- 장시간 야외 체류 회피
- 숙소 복귀 또는 중간 휴식 가능성

유모차 동반 가족:

- 계단 회피
- 화장실
- 수유 또는 휴식 가능성
- 주차 편의
- 좁은 길 또는 비포장 여부

고령자:

- 보행 부담
- 앉아서 쉴 공간
- 화장실
- 낙상 위험
- 날씨 영향

## 8. 결과 문장 원칙

점수 설명은 다음 구조를 따른다.

- 적합 이유: 사용자의 조건과 맞는 점
- 감점 이유: 부족하거나 확인이 필요한 점
- 확인 필요: 방문 전에 확인할 구체 항목
- 대체안: 조건이 맞지 않을 때 선택할 방향

금지:

- 단정적인 가능 표현
- 의료 효과 표현
- "무조건", "100%", "누구나" 표현
- 출처 없는 접근성 보장 표현

## 9. 점수 표시 예시

예시:

- 총점: 78점
- 등급: B
- 적합 이유: 도보 부담이 낮고 바다 감상 목적과 잘 맞습니다.
- 감점 이유: 장애인 화장실 운영 여부와 주차장-입구 거리는 방문 전 확인이 필요합니다.
- 확인 필요: 화장실 운영, 경사로 상태, 강풍 여부
- 정보 상태: 일부 확인
